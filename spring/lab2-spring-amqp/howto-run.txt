4. Running the Application
When you run the HelloWorldMessageApp class, the following will happen in order:

Spring Boot auto-configures the connection to RabbitMQ.

The Queue bean ensures that myQueue is created on the broker.

The run method executes, utilizing RabbitTemplate to send "Hello, world!" to myQueue.

The @RabbitListener detects the message on myQueue, consumes it, and prints Message read from myQueue : Hello, world! to your console.
